# Security Policy

## Reporting a vulnerability

If you discover a security vulnerability, please **do not** open a public issue.

Report privately via [GitHub Security Advisories](https://github.com/SunrisesIllNeverSee/signa/security/advisories/new).

## Scope

- The signa CLI and REPL
- The MCP server (12 tools, stdio transport)
- Local log reading and taste profile generation
- Privacy guarantees (local-only, token-only, Layer 3 opt-in)

## Out of scope

- The signalaf.com web app (report at [sigrank-app](https://github.com/SunrisesIllNeverSee/sigrank-app/security))
- The sigrank CLI/MCP server (report at [sigrank-mcp](https://github.com/SunrisesIllNeverSee/sigrank-mcp/security))

## Response timeline

- Acknowledgement within 48 hours
- Assessment within 7 days
